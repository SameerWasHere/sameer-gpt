Welcome to your projects workspace

This is a private, shared space where Claude drops things it builds for you to review — all behind a single PIN.

What lives here

Each tile on the workspace home is a self-contained HTML artifact: a mockup, a tool, a report, a prototype — whatever Claude put together. Tap a tile to open it full-screen.

How new artifacts get added

  1. Claude drops an HTML file into public/projects/artifacts/
  2. It adds an entry to public/projects/manifest.json with a title, description, category, and date
  3. The new tile shows up automatically on the workspace home

This page is the example

It's the first artifact, here so the whole system is testable end-to-end. Once real artifacts start landing, this can stick around or be removed from the manifest — your call.
